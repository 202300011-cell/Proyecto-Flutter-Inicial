# \## Descripción del proyecto

# 

# Columna es una aplicación desarrollada utilizando Flutter y Dart como parte del proyecto de la materia.

# 

# El objetivo del proyecto fue integrar en una sola aplicación diferentes conocimientos relacionados con el desarrollo de aplicaciones móviles y multiplataforma.

# 

# Durante el desarrollo se trabajó con interfaces gráficas, navegación entre pantallas, controles básicos y personalizados, estilos, temas, almacenamiento local, archivos, consumo de APIs, operaciones CRUD, programación asíncrona y notificaciones del sistema.

# 

# La aplicación fue desarrollada de manera modular, separando los modelos de datos y los servicios que realizan operaciones específicas. Esto permite que el proyecto sea más fácil de organizar, mantener y ampliar en el futuro.

# 

# El proyecto también fue probado en un dispositivo Android físico, con el objetivo de comprobar que la aplicación funcionara fuera del entorno de desarrollo.
